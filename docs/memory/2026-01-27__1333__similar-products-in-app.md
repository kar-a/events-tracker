# Задача 1333: Похожие товары в приложении

**Дата:** 2026-01-27  
**Задача:** 1333  
**Тип:** Фича приложения, YML-экспорт, Похожие товары

---

## Context

**Задача:** Реализовать отображение блока «Похожие товары» на детальной карточке товара в мобильном приложении (ImShop). Данные для похожих товаров уже есть: свойство каталога SIMILAR_PRODUCTS (заполняется кроном по фото, задача 1327), на сайте блок выводится в component_epilog карточки. Приложение получает контент через выгрузку YML; в обработчике экспорта оффера логика getSimilarProducts() и запись в тег rec уже были добавлены, но экспорт блокировался отладочным кодом и формат записи был неверным.

**Ограничения:** Формат rec должен совпадать с глобальными sliders в YML (items — список id элементов; title — строка).

---

## Key Decisions

- Удалить отладочный блок `pre([...]); die();` в onExportOfferWriteData.php, чтобы блок похожих товаров попадал в выгрузку.
- Использовать возвращаемое значение getSimilarProducts() корректно: метод возвращает `['items' => array of ID, 'title' => 'Похожие категория']`. В rec записывать `implode(',', $similarData['items'])`, в атрибут title — `$similarData['title'] ?? 'Похожие товары'`.
- Имя переменной заменено на $similarData для ясности (структура с keys items/title).

---

## Code Touchpoints

- **app/local/php_interface/lib/events/yandexmarket/onExportOfferWriteData.php**
  - Секция «Похожие товары (слайдер на детальной в приложении)»: удалены вызовы pre() и die(); при наличии $similarData формируется тег rec с содержимым `implode(',', $similarData['items'])` и атрибутом title из $similarData['title'] или «Похожие товары».
  - getSimilarProducts($element, $limit = 9): возвращает массив с ключами items (ID элементов) и при наличии категории — title; фильтрация по SIMILAR_PRODUCTS (getSimilarByPhoto), исключение дублей по IMAGES_HASH, лимит 9.

---

## Cause–effect

- **Причина:** В приложении на детальной не было блока похожих товаров; в YML логика была, но не работала из-за отладки и неверного формата.
- **Следствие:** После правок экспорт пишет в YML блок rec с похожими товарами; приложение получает данные и показывает слайдер «Похожие товары» на детальной.

---

## Verification

- Пересобрать/проверить выгрузку YML для ImShop: в офферах должен быть тег rec с title «Похожие …» и списком id через запятую.
- В приложении открыть детальную карточку товара с заполненным SIMILAR_PRODUCTS — должен отображаться слайдер похожих товаров.
