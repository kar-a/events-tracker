# 1355: Обновление правил автоблокировщика

Date: 2026-03-20
Task Key: 1355
Title: Обновление правил автоблокировщика

## Context
- Автоблокировщик иногда даёт ложные срабатывания при загрузке кэшированной версии стилей.
- Причина: в `app/local/php_interface/config/blocker.php` в секции `requestStarts` включён паттерн `/bitrix/cache`.

## Key Decisions
- Удалить паттерн `/bitrix/cache` из `requestStarts`, чтобы исключить блокировку штатных запросов к кэш-ассетам Bitrix.

## Code Touchpoints
- `app/local/php_interface/config/blocker.php`
  - `scanChecking.requestStarts`: удалён элемент `'/bitrix/cache' => ''`.

## Gotchas (Pitfalls)
- Нельзя удалять слишком «широкие» паттерны, которые защищают от реальных векторов атак; удаляем строго `/bitrix/cache` как источник ложных срабатываний при загрузке стилей.

## Verification
- Проверить, что при обычной загрузке страниц не происходит рост блокировок `_blockScanners_requestStarts`.
- Убедиться, что фронтенд корректно подхватывает кэшированные ассеты (CSS/JS/стили).

## Follow-ups
- При повторении ложных срабатываний — уточнить условия/паттерны в `requestStarts` или добавить исключения точнее под существующие URL ассетов.
