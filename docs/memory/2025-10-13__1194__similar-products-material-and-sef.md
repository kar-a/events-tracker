# Task Memory Card

Date: 2025-10-13
Task Key: 1194
Title: Материал в параметрах для похожих товаров + нормализация SEF

## Context
- Продолжение 1164: на детальной при `!CAN_BUY` показывается кнопка «Похожие …».
- Для часов не было ссылок из‑за отсутствия BRAIDING; требовалось учитывать MATERIAL.
- SEF‑путь формировался с подкатегорией, которая для некоторых категорий удаляется при очистке дерева.

## Key Decisions
- В `Catalog\Helper::getSimilarProductsLink` добавлен `MATERIAL` к параметрам сходства (вместе с `BRAIDING`).
- Базовый `SEF_FOLDER` собирается от категории; подкатегория добавляется только если реально существует в очищенном дереве `getCategoriesTree(false, true)`.

## Code Touchpoints
- Files → `app/local/php_interface/lib/app/Catalog/Helper.php`
  - `getSimilarProductsLink($product)`: MATERIAL в фильтре и проверка наличия подкатегории по очищенному дереву.

## Gotchas
- Не собирать URL вручную — ссылки формировать через `Catalog\Helper::getLinkByParams()` (каноничный порядок и алиасы).
- Подкатегория у некоторых категорий (например, «часы») может быть удалена функцией очистки; проверять по дереву перед добавлением в путь.

## Verification
- Недоступный товар категории «часы» → кнопка ведёт на `/catalog/chasy/` с применённым `MATERIAL`.
- Для товаров с BRAIDING поведение не меняется; фильтр добавляет оба параметра, если они есть.
