# Task Memory Card

Date: 2026-01-23
Task Key: 1300
Title: Убрать подчеркивание при переключении цвета в приложении

## Context
В мобильном приложении ImShop при переключении цвета товара у недоступных вариантов отображалась пунктирная обводка (подчеркивание). Это происходило из-за того, что в YML-выгрузке для ImShop недоступные варианты цвета помечались атрибутом `type='inconsistent-link'`, который использовался приложением для визуального выделения недоступных вариантов.

## Problem
- При переключении цвета в конструкторе товара в мобильном приложении недоступные варианты отображались с пунктирной обводкой (подчеркиванием)
- Это создавало визуальный шум и ухудшало пользовательский опыт
- Атрибут `type='inconsistent-link'` добавлялся в YML-выгрузку для всех недоступных вариантов цвета

## Solution
Убрано добавление атрибута `type='inconsistent-link'` для недоступных вариантов цвета в YML-выгрузке:
- Закомментированы строки, которые добавляли атрибут `type='inconsistent-link'` для недоступных значений цвета
- Добавлены комментарии `@deprecated (dashed обводка)` с объяснением причины удаления
- Изменения применены в двух местах: для текущего выбранного значения и для альтернативных значений

## Key Decisions
1. **Удаление визуального различия** — все варианты цвета теперь отображаются одинаково, без визуального различия по доступности
2. **Сохранение функциональности** — недоступные варианты остаются кликабельными и ведут на доступную конфигурацию
3. **Обратная совместимость** — код закомментирован, а не удален, что позволяет легко вернуть функциональность при необходимости
4. **Единообразие с веб-версией** — решение соответствует подходу в веб-версии (задача 1094c), где также убрана пунктирная обводка у недоступных цветов

## Code Touchpoints

### Backend
- `app/local/php_interface/lib/events/yandexmarket/onExportOfferWriteData.php`:

  - Строки 273-274: закомментировано добавление атрибута `type='inconsistent-link'` для альтернативных значений цвета (если они недоступны)
  - Добавлены комментарии `@deprecated (dashed обводка)` с объяснением

## Effects
- Убрано подчеркивание (пунктирная обводка) у недоступных вариантов цвета в мобильном приложении
- Все варианты цвета отображаются одинаково, без визуального различия по доступности
- Улучшен пользовательский опыт при переключении цвета в конструкторе товара
- Единообразное поведение с веб-версией сайта

## Intent
