Title: Слайдеры на главной

## Context
На главной странице сайта отображаются баннеры через компонент `app:banners` с шаблоном `slider`. Слайдер баннеров инициализируется в `component_epilog.php` через вызов метода `App.initSwiper()`, но слайдер не работал - не происходило переключение баннеров, не работали кнопки навигации и пагинация.

## Problem
Метод `App.initSwiper()` является асинхронным (использует `Router.assets.Swiper().then(...)` для загрузки библиотеки Swiper), но в `component_epilog.php` он вызывался синхронно через строку JavaScript, которая выполнялась сразу. Это приводило к тому, что когда скрипт выполнялся, библиотека Swiper еще не была загружена, и инициализация слайдера не происходила. В результате слайдер баннеров на главной странице не работал.

## Solution
Исправлена инициализация слайдера баннеров: вызов метода `App.initSwiper()` обернут в `.then()` для ожидания загрузки библиотеки Swiper перед инициализацией слайдера.

Изменения:
- В `component_epilog.php`: изменен синхронный вызов `App.initSwiper(...)` на асинхронный с использованием `.then()`: `App.initSwiper('#slider_main', {...}).then(() => { ... });`
- Это гарантирует, что слайдер инициализируется только после загрузки библиотеки Swiper

## Key Decisions
- Использовать `.then()` для ожидания загрузки библиотеки Swiper — это соответствует асинхронной природе метода `initSwiper` и гарантирует, что слайдер инициализируется только после загрузки библиотеки
- Сохранить логику инициализации в `component_epilog.php` — это стандартное место для инициализации компонентов в Bitrix
- Аналогичное решение с задачей 1282 — обе задачи решают проблему несоответствия между асинхронными методами и синхронными вызовами

## Code Touchpoints

## Effects
- Слайдер баннеров на главной странице корректно инициализируется и работает
- Переключение баннеров функционирует правильно
- Кнопки навигации (вперед/назад) работают корректно
- Пагинация слайдера функционирует правильно
- Улучшен пользовательский опыт на главной странице

## Intent
Исправить проблему с инициализацией слайдера баннеров на главной странице, которая возникла из-за несоответствия между асинхронной природой метода `initSwiper` и синхронным вызовом. Теперь слайдер корректно инициализируется после загрузки библиотеки, что восстанавливает функциональность переключения баннеров и улучшает пользовательский опыт на главной странице.
