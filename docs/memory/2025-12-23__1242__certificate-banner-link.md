# Task Memory Card

Date: 2025-12-23
Task Key: 1242
Title: Тест баннера с переходом на сертификаты на сайт в приложении

## Context
- Требовалось создать ссылку со внешнего источника (например, из баннера в мобильном приложении) на покупку онлайн сертификатов, которая автоматически открывала бы нужную вкладку формы и прокручивала к заголовку "Заказать сертификат".
- Страница сертификатов (`/certificates/`) имеет две вкладки: "с доставкой в красивом конверте" (`delivery-tab`) и "онлайн на E-mail" (`online-tab`).
- Вкладки реализованы через Bootstrap Tabs API (`data-bs-toggle="tab"`).

## Key Decisions
- URL-параметры для управления вкладками: `?tab=online` и `?tab=delivery`.
- Якорь для прокрутки: добавлен `<div id="certificate-order-title"></div>` перед заголовком формы.
- Универсальная реализация через маппинг параметров URL на ID вкладок в `AppCertificateForm.handleUrlTabParameter()`.
- Прокрутка выполняется после открытия вкладки (событие `shown.bs.tab`) с небольшой задержкой (100ms) для стабильности.
- Использование `{once: true}` для одноразового обработчика события прокрутки.

## Code Touchpoints
- `app/certificates/index.php` — добавлен якорь `<div id="certificate-order-title"></div>` перед заголовком "Заказать сертификат".
- `app/local/templates/trimiata/js/AppCertificateForm.js`:
  - Метод `handleUrlTabParameter()` — проверяет URL-параметр `tab`, маппит на ID вкладок, открывает через Bootstrap Tab API, прокручивает к якорю.
  - Маппинг: `'online' → 'online-tab'`, `'delivery' → 'delivery-tab'`.
  - Вызывается в `initEvents()` после инициализации формы.

## Gotchas
- Проверка наличия `window.bootstrap` и `window.bootstrap.Tab` перед использованием Bootstrap API.
- Задержка в 100ms перед прокруткой для обеспечения полного рендеринга вкладки.
- Использование `setTimeout` внутри обработчика события для плавной прокрутки.
- Важно регистрировать обработчик события `shown.bs.tab` до вызова `tabInstance.show()`.

## Verification
- Открыть `/certificates/?tab=online` — должна открыться вкладка "онлайн на E-mail" и прокрутиться к заголовку.
- Открыть `/certificates/?tab=delivery` — должна открыться вкладка "с доставкой в красивом конверте" и прокрутиться к заголовку.
- Без параметра `tab` — страница работает как обычно, без автоматического переключения вкладок.

## Follow-ups
- Возможность расширения функционала для других параметров (например, автозаполнение формы).
- При необходимости можно добавить поддержку хэша в URL (`#certificate-order-title`) как альтернативу или дополнение к параметру `tab`.
