## Context

Задача 1175: Быстрая покупка. Требуется отобразить `app:catalog.element` в компактном режиме в сайдбаре (`data-toggle="ajax-aside"`) и в мобильной панели `CupertinoPane`, открывая по клику на `data-role="product-show-offers"`. Внутри блока должны работать все JS‑функции (выбор оффера/веса, корзина, слайдер изображений, zoom/Fancybox).

## Key Decisions

- Создан API `app/api/modal/product-quick-buy.php`, который рендерит `app:catalog.element` с параметрами `COMPACT_MODE=Y`, `QUICK_BUY=Y`. Заголовок offcanvas скрывается (`title=''`).
- В `AppBasket.js` обработчик `elements.showOffers` переключён на `app.loadAjaxAside('product-quick-buy','api', {elementId, template})`.
- В `catalog.element/main` реализована поддержка `COMPACT_MODE`:
  - В `result_modifier.php` фильтр табов до одного ('Характеристики'); подготовка `IMAGES_LIST` (SRC/SRCSET/SIZES/ALT/TITLE) по паттерну задачи 1133.
  - В `template.php` добавлен класс `product_compact`, скрыт основной слайдер, оставлены `thumbnails` с Fancybox; миниатюрам добавлены `data-srcset`/`data-sizes`.
  - В `component_epilog.php` скрыты блоки «Комплекты», «Недавно просмотренные», `product__whatsapp_link`, `product__delivery_container`.
- В `App.js` инициализации для динамического DOM переведены на `$.initialize` для: детального слайдера (`[data-role=detail-image-slider]`) и zoom Fancybox, чтобы они работали внутри `CupertinoPane`.
- Для мобильной панели добавлен класс контейнера `mobile-pane__container__quick_buy`; на родитель `.pane` навешивается `pane_fast_buy` (реализовано через события `CupertinoPane`/добавлено проектом).

## Code Touchpoints

- JS: `app/local/templates/trimiata/js/AppBasket.js`, `app/local/templates/trimiata/js/App.js`.
- API: `app/api/modal/product-quick-buy.php`.
- Component: `app/local/templates/trimiata/components/app/catalog.element/main/{result_modifier.php,template.php,component_epilog.php}`.
- Helpers: `App\Template\Helper::IMAGE_SIZES`, `getListImageSizes()`.

## Gotchas

- Запрещено дублировать подключение `api/core.php` в модальных эндпоинтах: ядро уже подхватывается через `app/api/modal/index.php`. Иначе — `E_COMPILE_ERROR` (дублирование `ApiResult`).
- Для корректной работы слайдера/zoom в динамическом контенте используем `$.initialize`, иначе обработчики не навешиваются в пределах `CupertinoPane`.
- В `COMPACT_MODE` не подгружаем тяжёлые ресурсы (preload видео/больших изображений) и не показываем тяжёлые блоки (комплекты/просмотренные/whatsapp/delivery).
- Миниатюры должны использовать такой же подход `srcset/sizes`, как в задаче 1133 (ItemList/карточки). Для этого собран `IMAGES_LIST` в `result_modifier.php`.

## Verification

- Desktop offcanvas: клик по `data-role="product-show-offers"` → сайдбар с компактной карточкой; миниатюры кликабельны (Fancybox), цена/вес/оффера и корзина работают локально.
- Mobile pane: добавляются классы `mobile-pane__container__quick_buy` и `pane_fast_buy`; инициализируются слайдер и zoom; скролл/закрытие корректны.
- SEO/URL: Каноника не меняется, нормализация URL в BeforeProlog нетронута.
