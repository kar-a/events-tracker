# Task Memory Card

Date: 2026-01-10
Task Key: 1277
Title: Вывод суммы накоплений клиента

## Context
В личном кабинете пользователя на странице профиля отображалась сумма накоплений по карте лояльности, но это было некорректно, так как не отражало общую сумму накоплений клиента. Сумма накоплений клиента хранится в поле пользователя `UF_WASTED_MONEY` (потраченная сумма), которое учитывает все покупки клиента, а не только по конкретной карте.

## Problem
В личном кабинете отображалась сумма накоплений по карте лояльности, что было некорректно, так как не отражало общую сумму накоплений клиента. Это могло вводить пользователей в заблуждение относительно их реального прогресса в программе лояльности, особенно если у клиента было несколько карт или карта была заменена.

## Solution
Заменен источник данных для отображения суммы накоплений: вместо суммы по карте теперь используется сумма по клиенту из поля `UF_WASTED_MONEY`:
- В компоненте `main.profile.info` изменен источник данных для суммы накоплений: теперь используется поле `UF_WASTED_MONEY` клиента вместо суммы по карте
- Форматирование суммы накоплений выполняется с использованием `number_format()` для читаемого отображения
- В шаблоне компонента обновлен вывод суммы накоплений в блоке с картой лояльности

Изменения:
- В `component.php`: изменен источник данных для суммы накоплений с суммы по карте на сумму по клиенту (`UF_WASTED_MONEY`)
- В `template.php`: обновлен вывод суммы накоплений для отображения суммы по клиенту вместо суммы по карте

## Key Decisions
- Заменить источник данных с суммы по карте на сумму по клиенту (`UF_WASTED_MONEY`) — это обеспечивает корректное отображение общей суммы накоплений клиента, которая учитывает все покупки
- Использовать поле `UF_WASTED_MONEY` для отображения суммы накоплений — это поле содержит актуальную информацию о потраченной сумме клиента по всем картам
- Форматировать сумму с разделителями тысяч пробелами — улучшает читаемость больших сумм
- Сохранить размещение вывода суммы накоплений в блоке с картой лояльности — логически связано с программой лояльности

## Code Touchpoints
- `app/local/components/account/main.profile.info/component.php` — изменен источник данных для суммы накоплений с суммы по карте на сумму по клиенту (`UF_WASTED_MONEY`)
- `app/local/components/account/main.profile.info/templates/.default/template.php` — обновлен вывод суммы накоплений для отображения суммы по клиенту

## Effects
- Исправлено некорректное отображение суммы накоплений: теперь показывается сумма по клиенту вместо суммы по карте
- Пользователи видят точную общую сумму своих накоплений, которая учитывает все покупки
- Улучшена корректность данных в личном кабинете
- Пользователи могут правильно оценивать свой прогресс в программе лояльности

## Intent
Исправить некорректное отображение суммы накоплений: заменить вывод суммы по карте на сумму по клиенту для обеспечения корректности данных. Теперь пользователи видят точную общую сумму своих накоплений, что помогает им правильно оценивать свой прогресс в программе лояльности.
