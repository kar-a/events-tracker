# ClickHouse (DDL и данные)

## Где лежит SQL

- **Миграции / init:** `infra/clickhouse/sql/` — файлы `01_*`, `02_*`, `03_*` и далее по мере роста.
- **Применение вручную (после `make up`):** `scripts/init-clickhouse.sh` (обходит контейнер `trimiata-events-clickhouse`).
- **Авто-init при первом старте контейнера:** том смонтирован в `docker-entrypoint-initdb.d` (см. `infra/compose/docker-compose.events.yml`).

## Соглашения

- Префиксы номеров: сейчас `01_`, `02_`, `03_*`; при большом числе схем можно перейти на `001_`, `010_`.
- Новые таблицы — только сюда, без копий в других каталогах репозитория.
